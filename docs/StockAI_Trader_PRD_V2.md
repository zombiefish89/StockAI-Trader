# StockAI Trader 产品需求文档 (V2)

## 概述
StockAI Trader 是面向普通股票交易者的 AI 投资副驾，帮助用户更快看懂个股、更快发现市场机会、更好管理风险，而不是直接替用户做决算。工具聚焦“看懂 + 比较 + 验证 + 计划”四个核心环节。

## 核心模块

### 单股 AI 仪表盘
提供一眼看懂某只股票的综合视图，包括：
- **公司快照**：业务概览、资务趋势（收入、利润、现金流）、业务构成等。
- **估值与历史对比**：当前估值指标（PE、PS、PB 等）在过去 3-5 年和行业中的分位。
- **技术面解读**：多时间框架下的趋势判断、关键支撑阶位、波动率信息。
- **重要事件时间轴**：最近一段时间的新闻、公告和资报的关键事件摘要。
- **AI 总结与预测**：结合以上信息，AI 用自焵语言总结公司现状、潜在机会与风险，并给出短期涨跌分析预测。预测背后会引用历史数据作为证据支撑。
