# StockAI Trader 产品需求文档（V2.1）

面向普通交易者的 AI 投资副驾

---

## 产品定位（Positioning）

StockAI Trader 是一款面向普通股票投资者的 **AI 投资副驾（Investment Copilot）**。

它不下单、不提供交易指令，而是让用户可以轻松完成完整的投资循环：

```
发现 → 看懂 → 比较 → 计划 → 复盘
```

其中 **证据级回测（Evidence Backtesting）** 作为隐藏式底层能力，自动支撑 AI 的判断，让所有分析更可信。

目标：让普通用户每天 5–15 分钟，也能做出高质量、体系化的投资决策。

---

# 1. 发现（Discover）

帮助用户快速了解市场情况、找到值得关注的股票。

模块包含：

```
1.1 市场热点雷达（Themes Radar）
1.2 宏观与新闻监控（Macro & News Monitor）
1.3 个性化股票推荐（Personalized Discovery）
1.4 观察列表（Watchlist）
```

## 1.1 市场热点雷达

- 热点行业涨跌榜（AI、半导体、能源等）
- 热点主题聚类（AI Infra、存储、电动车等）
- 领涨领跌股票
- 市场温度（偏热 / 中性 / 偏冷）
- 主题轮动提示
- AI 今日一句话总结

## 1.2 宏观与新闻监控

- 美联储讲话摘要（鹰/鸽）
- 重要数据：CPI / PCE / 就业 / GDP
- 行业政策事件
- 全球市场趋势（纳指/标普/VIX/BTC）
- AI 自动生成宏观摘要

## 1.3 个性化股票推荐

推荐来源：

- 用户持仓 → 相似公司与替代品
- 用户历史关注 → 相似商业模式
- 当前市场热点 → 用户未关注的主题标的
- 基本面筛选 → 高成长、估值合理公司

输出结构：

- 推荐理由（AI 解释）
- 基本面简述
- 技术面强弱
- 所属主题标签

## 1.4 观察列表（Watchlist）

用户可随时将股票加入观察列表，便于后续「看懂」「比较」。

# 2. 看懂（Understand）

提供一只股票的完整视图。

包括：

```
2.1 单股 AI 仪表盘
2.2 财报与估值分析
2.3 技术结构
2.4 事件时间轴
2.5 【隐藏】证据级回测
```

## 2.1 单股 AI 仪表盘

### ● 公司快照

- 业务简介
- 市值 / 收入 / 利润 / 现金流趋势
- 业务结构

### ● 估值结构

- PE / PS / PB / EV
- 估值分位（3/5 年）
- 行业对比
- AI 判断（便宜/合理/偏贵）

### ● 技术结构

- 趋势方向
- 支撑/阻力
- 波动率 ATR
- 周线 vs 日线 强弱

### ● AI 综合结论（核心）

- 短期 & 中期趋势预测（含置信度）
- 机会点 / 风险点
- 逻辑解释（基本面、估值、技术、新闻）

## 2.2 财报与估值分析

- 最近 8 季营收/利润趋势
- 指引摘要
- 亮点与风险点
- 财报周期典型波动（自动回测）

## 2.3 技术结构

- 趋势阶段识别
- 多周期强弱
- 自动支撑/阻力
- 高频形态识别（突破/下破）

## 2.4 事件时间轴

- 财报
- 重磅新闻
- 产品发布
- 合作/监管
- 股价异动背后原因

## 2.5 隐式：证据级回测（自动触发）

- 胜率
- 收益分布
- 最大回撤
- 历史触发点图
- 可靠性解释

# 3. 比较（Compare）

帮助用户在多只股票间做选择。

```
3.1 对比篮（Compare Basket）
3.2 持仓自动对比
3.3 AI 自然语言对比
```

## 3.1 对比篮

- 用户可选择 2–4 只股票
- 一键生成对比报告

对比内容：

- 基本面
- 估值
- 技术结构
- 风险
- 成长 vs 稳定对比
- AI 综合总结

## 3.2 自动持仓对比

系统自动检测：

- 同行业集中
- 高频相关股票
- 潜在替代品

## 3.3 AI 自然语言对比

用户可问：

- 比较 META 和 APP
- NVDA 和 AMD 哪个更稳？
- 生成我持仓里科技股的对比

# 4. 计划（Plan）

帮助用户规划仓位与风险。

包括：

```
4.1 持仓结构分析
4.2 风险暴露分析
4.3 加仓/减仓模拟
4.4 财报事件提醒
```

## 4.1 持仓结构分析

- 单票集中度
- 行业/国家暴露
- 波动风险分布
- 高估值暴露
- AI 风险总结

## 4.2 风险暴露分析

- 主题暴露
- 高波动比例
- 财报密集风险
- 高相关股票风险

## 4.3 加仓/减仓模拟

模拟后自动计算：

- 行业暴露变化
- 风险变化
- 稳健/成长度变化
- AI 提示（非交易）

## 4.4 财报与事件提醒

提醒：

- 公司财报
- 重要政策
- 行业会议
- 宏观数据（CPI / 非农 / FOMC）

---

# 5. 复盘（Review）

让用户从行为层面成长。

包括：

```
5.1 交易行为分析
5.2 盈亏结构分析
5.3 每周 AI 投资体检
5.4 行为模式分析（未来）
```

## 5.1 交易行为分析

识别行为：

- 追高
- 割肉
- 押方向
- 高频交易
- 卖飞

AI 输出：

- 行为倾向
- 优势
- 风险
- 建议

## 5.2 盈亏结构分析

展示：

- 盈利来源
- 亏损集中点
- 最大回撤来源
- 最成功的操作类型

## 5.3 每周 AI 投资体检

包含：

- 本周表现
- 组合结构变化
- 风险暴露
- 盈亏分析
- 行为偏差
- 下周事件

## 5.4 行为模式分析（未来）

- 主题偏好
- 波动偏好
- 比较习惯
- 研究 vs 决策路径
